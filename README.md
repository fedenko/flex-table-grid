# flex-table-grid
Flexbox grid width table fallback.

## Why?
Main idea was to build responsive grid system that will be supported by mobile devices, Internet Explorer and WeasyPrint.

## Build

1. Install node.js
2. Install gulp
3. Install development dependencies `$ npm install`
4. Run `$ gulp dist`
